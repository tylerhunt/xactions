# Implementation Plan: UI Redesign — Remove DaisyUI

**Branch**: `004-ui-tailwind-redesign` | **Date**: 2026-04-10 | **Spec**: [spec.md](spec.md)

## Summary

Restyle all non-budget pages and shared components to match the design system established on the budget page. Remove all DaisyUI component class names (`btn`, `card`, `stat`, `badge`, `alert`, `modal`, etc.) from templates. Fix contrast issues caused by the app background changing from dark to warm off-white.

No new data models, routes, or backend logic. This is a purely presentational change across the Phoenix LiveView templates and shared component modules.

## Technical Context

**Language/Version**: Elixir 1.17+ / OTP 27
**Primary Dependencies**: Phoenix 1.8.5, Phoenix LiveView 1.1.x, Tailwind CSS (arbitrary values), Heroicons
**Storage**: N/A — no data model changes
**Testing**: ExUnit + Phoenix.LiveViewTest

**Performance Goals**: Page render time unchanged; no new JS or asset changes
**Constraints**: No DaisyUI component classes in rendered HTML after this change; WCAG AA contrast (4.5:1) on all text; `mix test` must pass at 0 failures throughout
**Scale/Scope**: 7 live views + 3 shared components + 1 layout component

### Design Patterns (from research.md)

**Buttons**:
- Primary: `px-4 py-2 bg-[#030213] text-white rounded-lg text-sm hover:bg-[#030213]/90 transition-colors`
- Ghost: `px-4 py-2 hover:bg-[#ececea] rounded-lg text-sm transition-colors text-[#717182] hover:text-[#030213]`
- Danger link: `text-xs text-[#717182] hover:text-[#d4183d] transition-colors`
- Toggle active: `px-3 py-1.5 rounded-lg text-sm bg-[#ececea] text-[#030213]`
- Toggle inactive: `px-3 py-1.5 rounded-lg text-sm text-[#717182] hover:bg-[#ececea]/50 transition-colors`

**Cards**: `bg-white border border-black/[.08] rounded-xl` with `p-5` or `p-4` padding

**Inputs**: `w-full border border-black/[.08] rounded-lg px-3 py-2 text-sm`

**Alert banners**:
- Error: `border-l-4 border-[#d4183d] bg-[#d4183d]/5 rounded-lg px-4 py-3 text-sm`
- Warning: `border-l-4 border-[#f59e0b] bg-[#f59e0b]/5 rounded-lg px-4 py-3 text-sm`

**Status badges** (pill): `text-xs font-medium px-2 py-0.5 rounded-full` + semantic `bg-*/10 text-*` colors

**Stat/summary**: `bg-white border border-black/[.08] rounded-xl p-5` grid item with `text-sm text-[#717182] mb-1` label + `text-2xl tracking-tight` value

**Page root**: `min-h-screen bg-[#f8f7f5]`
